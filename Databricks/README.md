 # Databricks & Terraform

This folder contains the Databricks and Terraform assets.

## Contents

### `bundles`
Declarable Automation Bundles

### `notebooks`
Jupyter Notebooks and Python files. Most noteably, the `modules` folder that contains the reusable wheels for Terraform Modules

### `terraform`
Terraform scripts for infrastructure provisioning and management. Most noteably contains the [:star:`scalable_ingestion`](/scalable_ingestion) terraform project.
