# resources/ — generated

The per-domain `*.gen.yml` files here (one per domain: bronze schema + checkpoint
volume + ingestion job) are **generated by Terraform**
(`Databricks/terraform/lakeflow_connect/ingestion`) from `terraform.tfvars`, so
the domain list stays single-sourced. See [`job.yml.tftpl`](../../../modules/domain_ingest/templates/job.yml.tftpl) for the template

Do not hand-edit or commit them. To add/change a domain, edit the `domains` map in
the ingestion root's `terraform.tfvars` and run `terraform apply`, then
`databricks bundle deploy`.
